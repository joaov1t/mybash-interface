# 💻 MyBash - interface

Repositório criado para deixar seu terminal ZSH mais bonito e personalizado.

Eu fiz inicialmente para uso próprio, mas resolvi deixar público para demonstrar na prática alguns conceitos de Bash Script e customização de ambiente.

---

## 🚀 Instalação

```bash
git clone https://github.com/joaov1t/mybash-interface && cd mybash-interface
```

```bash
 sudo cp -r mybash-interface.sh ~/.zshrc && cp -r mybash-interface.sh ~/.zshrc
```
```bash
chsh -s $(which zsh)
```

---

## ⚙️ O que esse projeto faz

* Customiza o terminal ZSH
* Adiciona melhorias visuais
* Deixa o ambiente mais organizado e produtivo
